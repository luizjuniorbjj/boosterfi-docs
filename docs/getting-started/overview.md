# Getting Started

Welcome to BoosterFi! This guide will help you start earning real yield from Flash Loans.

---

## What You'll Need

Before you begin, make sure you have:

- [x] A Web3 wallet (MetaMask recommended)
- [x] USDC on Arbitrum One network
- [x] Small amount of ETH on Arbitrum for gas fees

---

## Quick Start

### Step 1: Set Up Your Wallet

1. Install [MetaMask](https://metamask.io/) browser extension
2. Create a new wallet or import existing one
3. **Keep your seed phrase safe!**

### Step 2: Add Arbitrum Network

Add Arbitrum One to your MetaMask:

| Parameter | Value |
|-----------|-------|
| **Network Name** | Arbitrum One |
| **RPC URL** | `https://arb1.arbitrum.io/rpc` |
| **Chain ID** | `42161` |
| **Currency Symbol** | ETH |
| **Block Explorer** | `https://arbiscan.io` |

!!! tip "Easy Setup"
    Visit [app.boosterfi.app](https://app.boosterfi.app) and click "Add Arbitrum" to automatically add the network.

### Step 3: Get USDC on Arbitrum

You can get USDC on Arbitrum by:

1. **Bridge from Ethereum** — Use [Arbitrum Bridge](https://bridge.arbitrum.io)
2. **Buy directly** — Purchase on exchanges that support Arbitrum withdrawals
3. **Swap on DEX** — Use Uniswap or other Arbitrum DEXs

### Step 4: Connect to BoosterFi

1. Visit [app.boosterfi.app](https://app.boosterfi.app)
2. Click "Connect Wallet"
3. Select MetaMask
4. Approve the connection

### Step 5: Make Your First Deposit

1. Enter deposit amount (minimum $100 USDC)
2. Select lock period (1, 5, 10, or 20 days)
3. Click "Deposit"
4. Confirm transaction in MetaMask
5. Done! You're now earning Flash Loan fees

## How It Works

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   YOU DEPOSIT  →  LIQUIDITY POOL  →  FLASH LOANS  →  FEES      │
│      USDC          BoosterFi         Executed       to YOU     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

1. **Deposit** — You deposit USDC into the protocol
2. **Pool** — Your capital joins the liquidity pool
3. **Flash Loans** — Traders and bots use the liquidity
4. **Fees** — Each Flash Loan generates 0.09% fee
5. **Earnings** — You receive 80% of fees proportionally

---

## Understanding Lock Periods

| Lock Period | Best For |
|-------------|----------|
| **1 Day** | Maximum flexibility, testing the protocol |
| **5 Days** | Short-term commitment |
| **10 Days** | Medium-term investment |
| **20 Days** | Long-term commitment |

!!! note "Important"
    You cannot withdraw your principal before the lock period ends. Yields can be claimed at any time.
